# 项目计划：港铁线路压力与人群流动模拟 (MTR Stress & Flow Simulation)

## 0. 项目目标
利用港铁开放数据（实时列车资讯 + 静态线网/车费数据），构建动态交通流模型，模拟不同时段乘客分布，识别线网拥塞瓶颈，并分析无障碍设施故障对人群流动的影响。

## 3. 核心算法详述 (Detailed Algorithms & ML Techniques)

本项目将传统交通模型与现代机器学习技术相结合，以增强模拟的准确性与挖掘深度。

### 3.1 预处理：空间聚类与特征提取 (Spatial Clustering)
*   **任务**: T2.3 权重归一化与特征提取。
*   **算法**: **K-Means 聚类**。
*   **描述**: 利用地政总署 POI 数据，对全港车站进行无监督聚类。通过车站周围建筑密度（办公、住宅、教育、休闲）将车站自动分类为“居住中心”、“商业中心”或“混合区”，作为引力模型中吸引力因子 ($A_i, B_j$) 的输入特征。

### 3.2 特征学习与预测：极端梯度提升树 (XGBoost)
*   **任务**: T3.1 需求生成 (OD 挖掘)。
*   **算法**: **XGBoost 回归模型 (Extreme Gradient Boosting)**。
*   **描述**: 将复杂的 OD 需求预测转化为监督式机器回归问题。输入特征包括：起终点 POI 聚类结果、乘车总时长、总车费、换乘次数等。XGBoost 能够极其高效、鲁棒地捕捉这些多维特征与空间客流之间的非线性关系，并输出高解释性的特征重要度 (Feature Importance)。

### 3.3 路径预测：多项 Logit 模型 (Multinomial Logit Model, MNL)
*   **任务**: T3.2 路径分配。
*   **算法**: **基于 Softmax 的离散选择模型**。
*   **描述**: 模仿分类神经网络的输出层，计算每条候选路径的选择概率。输入特征包括：路径总时长、总车费、转乘次数及实时延迟系数。公式为：$P(i) = \frac{e^{V_i}}{\sum e^{V_j}}$，其中 $V$ 是路径的效用函数。

### 3.4 压力分析：异常检测与图传播 (Graph Propagation)
*   **任务**: T4.2 & T5.2 瓶颈识别与级联失效。
*   **算法**: **图卷积网络 (GCN) 启发式传播 / PageRank 变体**。
*   **描述**: 将线路压力视为节点上的信号。当某站（如金钟）压力超载时，模拟压力如何沿着物理连接（Edges）向相邻车站传播。利用 **Isolation Forest (孤立森林)** 算法在时间序列压力数据中自动识别“异常拥塞时段”，以实现自动预警。

### 3.5 评估：消融实验与回归分析 (Regression Analysis)
*   **任务**: T5.1 & T5.3 验证与评估。
*   **算法**: **多元线性回归 (Multiple Linear Regression)**。
*   **描述**: 建立回归模型验证模拟流向与官方吞吐量的一致性。通过比较加入 POI 特征前后的 $R^2$ 评分提升，定量评估数据挖掘对提升模型解释力的贡献。

## 7. 课程教授评审建议 (Professor's Evaluation & Tips)


### 核心要素审核：
*   **挖掘深度**：通过 **POI 地理特征交叉 + XGBoost 机器学习回归模型** 挖掘潜在 OD 矩阵，属于高质量的知识提取。
*   **技术复杂度**：集成了空间缓冲区分析、多层有向图、Logit 随机分配等高级算法，展现了较强的技术水平。
*   **分工与协作**：任务依赖关系明确，分工均衡，能够体现团队合作。

### 教授加分建议 (Tips for Full Marks)：
1.  **可重复性 (Reproducibility)**：确保 `requirements.txt` 和 `README.md` 详尽。代码中需包含数据清洗的详细注释，以便助教复现。
2.  **验证环节 (Validation)**：在报告中加入挖掘结果与现实常识（如“金钟站是转乘瓶颈”）的对比，或者对比官方吞吐量数据，以证明挖掘出的 OD 矩阵的正确性。
3.  **模型解释力 (Interpretability)**：在报告中务必利用 XGBoost 的特征重要性 (Feature Importance) 绘制柱状图，直观解释不同变量（如“商业区POI”与“乘车距离”）对人流吸引的量化贡献，展现白盒机器学习模型的解释能力。
4.  **消融实验 (Ablation Study)**：展示“不加入 POI 权重”与“加入 POI 权重”后的压力模拟结果差异，证明数据挖掘对模型精度的显著提升。
